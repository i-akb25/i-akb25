<summary><b>◉ SYSTEM 01 — TITAN OS</b></summary>

<br>

## Industrial Edge Telemetry & Predictive AI Platform

```text
STATUS      : OPERATIONAL
CATEGORY    : Industrial AI / IIoT
VERSION     : 1.x
ARCHITECTURE: Hybrid Edge + Cloud
```

Titan OS is a closed-loop Industrial IoT platform designed to simulate heavy industrial environments and demonstrate predictive maintenance through real-time telemetry, cloud data warehousing, and intelligent fault detection.

Unlike a traditional dashboard, Titan OS combines **Edge Computing, Cloud Infrastructure, WebSockets, Docker, Python simulation, MySQL, and Business Intelligence** into a complete engineering ecosystem.

### System Overview

| Module | Description |
|---------|-------------|
| Edge Node | Python-based telemetry simulator generating live industrial data |
| Gateway | Express.js server streaming telemetry through WebSockets |
| Dashboard | React application visualizing machine health in real time |
| Data Warehouse | Cloud MySQL telemetry storage |
| Analytics | Power BI dashboards & historical trend analysis |
| Deployment | Dockerized multi-service architecture |

### Engineering Stack

```text
React
Node.js
Express
Python
Docker
MySQL
WebSockets
Power BI
```

### Engineering Highlights

- Real-time telemetry streaming
- Predictive Time-To-Failure estimation
- Hybrid Offline + Cloud Architecture
- Automated Fault Detection
- Historical Analytics Pipeline
- Docker Orchestration
- CSV Export Pipeline
- Edge Simulation Engine

**Repository**
